Forestry Inventory

Using LiDAR and Global Mapper, I classified different features into specific classes, such as vegetation, ground, and buildings. Based on these classified features, I created digital surface models and digital terrain models to produce a tree canopy height models. The tree canopy model was used to identify tree tops and delineate individual trees. To share the results, I designed and scripted a Python Streamlit dashboard to display an inventory of trees, including information on height, spread, and elevation. The dashboard is hosted on an AWS EC2 instance for audiences to view insights on what types of trees are in the surrounding area.

Evaluating Potential Wildlife Crossings in Southern California

The goal of this project is to analyze Southern California's fragmented habitats and to help reduce wildlife-vehicle conflicts by allowing a passageway for wildlife to cross anthropogenic barriers. This will help provide a solution for habitat fragmentation and reduce the number of collisions between vehicles and animals. With these wildlife crossings, animal species will be able to thrive and we will be able to provide a safer environment for both humans and wildlife.
