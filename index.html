Live Tree Inventory

This dashboard is continually updated with a database on AWS S3. The LiDAR data is fed into a pipeline where a digital elevation model (DEM) and digital surface model (DSM) are created to produce a canopy height model (CHM). This is later put into PyCrown script, which calculates the tree top positions and delineates individual tree crowns. This data is then pulled onto AWS S3 and fed into the dashboard, which is hosted on a EC2 on AWS.
